# ScalculatorSBT
A Scala command line calculator using shunting-yard algorithm (not very Scala-like), SBT and ScalaTest. 

## Example
>2 * ((3 * 2) + 5)
>22

>sqrt(2/5)^2 * abs(-1)
>0.4

## Supports
>parentheses
>operations +, -, *, /, ^ (exponentation)
>functions sqrt, abs, sin, cos, tan (one argument)
>range of values is that of Double
